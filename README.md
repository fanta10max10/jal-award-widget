# JAL・ANA 特典航空券 予約開始日ウィジェット

iPhoneのロック画面・ホーム画面に **JAL** と **ANA** の特典航空券予約開始日を表示するScriptableウィジェットです。
JAL用・ANA用を**別ウィジェットとして小サイズで並べて配置**するのが推奨スタイルです。

---

## 予約ルール早見表

| 航空会社 | 予約開始 | 開始時刻 |
|---|---|---|
| **JAL** | 搭乗 **360日前** | **0:00**（深夜・国内国際とも） |
| **ANA** | 搭乗 **355日前** | 国内線 **9:30** / 国際線 **9:00** |

> JALはANAより約5日早く予約開始。ただし深夜0時スタートのため、繁忙期は0時と同時に即完売することがある。

## インストール手順

### JALウィジェット

1. Scriptable アプリを開き、右上 **「+」** で新規スクリプト作成
2. `jal_award_widget.js` の内容を全てコピペ
3. スクリプト名を **「JAL予約開始日」** に変更して保存
4. **▶** で赤テーマのプレビューが出ることを確認

### ANAウィジェット

1. 同様に **「+」** で新規スクリプト作成
2. `ana_award_widget.js` の内容を全てコピペ
3. スクリプト名を **「ANA予約開始日」** に変更して保存
4. **▶** で青テーマのプレビューが出ることを確認

---

## ロック画面への配置（推奨レイアウト）

**2つの `accessoryRectangular` を縦に並べる**

1. ロック画面を**長押し** → 「カスタマイズ」 → 「ロック画面」
2. 時計下のウィジェット枠をタップ → **「Scriptable」** を選択
3. 1つ目に `JAL予約開始日` を設定
4. もう1つ枠を追加し `ANA予約開始日` を設定

> **表示例**
> ```
> ┌──────────────────────────────┐
> │ JAL予約開始日                │
> │ 05/01（金）                  │
> │ 0:00 スタート（国内・国際とも）│
> └──────────────────────────────┘
> ┌──────────────────────────────┐
> │ ANA予約開始日                │
> │ 04/26（月）                  │
> │ 国内 9:30 / 国際 9:00 スタート│
> └──────────────────────────────┘
> ```

---

## ホーム画面への配置

**2つの `small` ウィジェットを横に並べる**

1. ホーム画面を**長押し** → 左上 **「+」**
2. 「Scriptable」→「Small」→「ウィジェットを追加」
3. ウィジェットを長押し → 「ウィジェットを編集」 → `JAL予約開始日` を選択
4. 同様に2つ目を追加して `ANA予約開始日` を選択

## 繁忙期インジケーター

表示中の日付が繁忙期に該当すると自動でアラート表示されます。

| 繁忙期 | 範囲 | 表示例 |
|---|---|---|
| 年末年始 | 12/28〜1/4 | `🔥 年末年始` |
| GW | 4/29〜5/6 | `🔥 GW` |
| お盆 | 8/10〜8/18 | `🔥 お盆` |
| シルバーウィーク | 9/19〜9/23 | `🔥 シルバーウィーク` |

- **ロック画面**: 時刻行の末尾に `🔥 GW` を追記
- **ホーム画面**: 日付がオレンジに変色 ＋ `🔥 GW期間 ― 即完売注意` バッジを表示

## 定数の変更方法

ルール変更時は各スクリプト冒頭の定数を修正するだけです。

**`jal_award_widget.js`**
```javascript
const OFFSET_DAYS = 360;    // 予約開始日数
const START_TIME  = "0:00"; // 開始時刻
```

**`ana_award_widget.js`**
```javascript
const OFFSET_DAYS         = 355;   // 予約開始日数
const START_TIME_DOMESTIC = "9:30"; // 国内線開始時刻
const START_TIME_INTL     = "9:00"; // 国際線開始時刻
```

## トラブルシューティング

| 症状 | 対処 |
|---|---|
| ウィジェットが「Run Script」のまま | 長押し→編集→Scriptで該当スクリプトを選択 |
| 日付が更新されない | Scriptableを開いて▶で手動実行。iOSの更新タイミングは自動制御 |
| ロック画面に表示されない | iOS 16未満の可能性。設定→一般→ソフトウェアアップデート確認 |
| ANAがブラウザで開く | ANAアプリ未インストール。App StoreからANAアプリをインストール |

---

## 注意事項

- iOSウィジェットは厳密にリアルタイム更新されません。実際の予約は各社アプリ/サイトの時計を基準にしてください。
- JALは **深夜0時ジャスト**、ANAは **9:30（国内）/ 9:00（国際）** から予約開始です。アラームのセットをお忘れなく。
