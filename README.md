# JAL特典航空券 予約開始日ウィジェット

iPhoneのロック画面・ホーム画面に「今日から360日後の日付（JAL特典航空券の予約可能最遠日）」を表示するScriptableウィジェットです。

---

## できること

- **ロック画面を見るだけで**「今日予約できる最も先の搭乗日」が即わかる
- 毎日0時に自動で繰り上がる（手動操作不要）
- ウィジェットをタップ → JAL特典航空券予約ページへ直接ジャンプ
- オフライン動作・追加コストなし（Scriptableは無料）

---

## 事前準備

1. App Storeから **Scriptable**（無料）をインストール
2. iOS **16以降**であることを確認（ロック画面ウィジェット対応）

---

## インストール手順

1. **Scriptable** アプリを開く
2. 右上の **「+」** をタップして新規スクリプト作成
3. `jal_award_widget.js` の内容を**全てコピー**してペースト
4. 左上のスクリプト名を **「JAL予約開始日」** に変更して保存
5. **▶（再生ボタン）** をタップしてプレビューが表示されることを確認

---

## ロック画面への配置（推奨）

1. ロック画面を**長押し** → 「カスタマイズ」 → 「ロック画面」
2. 時計下のウィジェット枠をタップ
3. 「**Scriptable**」を選択
4. **accessoryRectangular**（四角形）サイズを選んで配置
5. 配置後にウィジェットをタップ → `Script: JAL予約開始日` を選択
6. 「完了」→「設定」で確定

> **表示例**
> ```
> JAL予約開始日
> 2027/05/01
> 今日 5/6
> ```

---

## ホーム画面への配置（任意）

1. ホーム画面を**長押し** → 左上の「**+**」
2. 「Scriptable」を選択 → サイズを選んで「ウィジェットを追加」
3. ウィジェットを長押し → 「ウィジェットを編集」
4. Script に **JAL予約開始日** を選択

> **表示例（small）**
> ```
> JAL予約開始日
>
> 今日
> 2026/05/06
>
> +360日後（予約可能最遠日）
> 2027/05/01
>
> [✈ 今すぐ予約]
> ```
> ウィジェット全体（または「✈ 今すぐ予約」ボタン）をタップすると予約ページが開きます。

---

## 予約ページへのジャンプ（ボタン機能）

ウィジェットをタップすると、コード内の `BOOKING_URL` に設定されたJAL特典航空券予約ページが開きます。

### URLの有効期限について

コードに埋め込まれているURLには**セッショントークン**（`MESSAGE_AUTH`等）が含まれており、**期限切れになることがあります**。  
URLが期限切れになったら以下の手順で更新してください。

1. Safari で JAL公式サイトにログイン後、特典航空券予約ページに移動
2. アドレスバーのURLをコピー
3. Scriptableアプリでスクリプトを開き、`BOOKING_URL` の値を差し替えて保存

## ウィジェットサイズ一覧

| サイズ | 配置場所 | 表示内容 |
|---|---|---|
| `accessoryInline` | ロック画面・時計上1行 | `JAL: M/D` |
| `accessoryRectangular` | ロック画面・四角枠 | ラベル + 360日後 + 今日 |
| `accessoryCircular` | ロック画面・円形 | 月/日のみ |
| ホーム画面（small/medium/large） | ホーム画面 | 今日・360日後・予約ボタン |

---

## OFFSET_DAYS の変更方法

JALのルールが変更された場合は、スクリプト冒頭の定数を修正するだけです。

```javascript
const OFFSET_DAYS = 360;  // ← この数字を変更
```

> 参考: JGCプレミア・JMBダイヤモンド会員は国内線で **361日前 22:00** から先行予約可能。
> 該当ステータス取得後は `OFFSET_DAYS = 361` に変更し、別途時刻考慮が必要。

## トラブルシューティング

| 症状 | 対処 |
|---|---|
| ウィジェットが「Run Script」のままになる | ウィジェット長押し→編集→Script で `JAL予約開始日` を選択 |
| 日付が更新されない | iOSが更新タイミングを内部制御（数時間に1回）。手動で更新したい場合はScriptableアプリを開いて▶をタップ |
| ロック画面に表示されない | iOS 16未満の可能性。設定→一般→ソフトウェアアップデートを確認 |
| 予約ページが開かない | URLの期限切れの可能性。上記「URLの有効期限について」を参照 |

---

## 注意事項

- iOSウィジェットは厳密にリアルタイム更新されません（OSが管理するタイミングで更新）。深夜0時の切り替わりに数分〜数十分のラグが発生する場合があります。**実際の予約はJALサイトの時計を基準にしてください。**
- 360日はJALの2024年3月26日改定後のルールです。将来変更される可能性があります。
