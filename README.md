# JAL・ANA 特典航空券 予約開始日ウィジェット

iPhoneのロック画面・ホーム画面に **JAL** と **ANA** の特典航空券予約開始日を並べて表示するScriptableウィジェットです。

---

## 予約ルール早見表

| 航空会社 | 予約開始 | 開始時刻 | 備考 |
|---|---|---|---|
| **JAL** | 搭乗 **360日前** | **0:00**（深夜） | 上級会員先行なし |
| **ANA** | 搭乗 **355日前** | **9:30**（国内線）/ 9:00（国際線） | 上級会員先行は2025年廃止 |

> JALはANAより約5日早く予約開始。ただしJALは深夜0時ジャストスタートのため、GW・年末などの繁忙期は0時と同時に即完売することがある。

---

## できること

- ロック画面を見るだけで **JAL・ANA 両社の予約開始日**（曜日付き）が即わかる
- 毎日0時に自動で1日進む（手動操作不要）
- **「✈ JAL予約」ボタン** → JALアプリ（App Store経由）を起動
- **「✈ ANA予約」ボタン** → ANAアプリの予約画面（Universal Link）を起動
- オフライン動作・追加コストなし（Scriptableは無料）

---

## 事前準備

1. App Storeから **Scriptable**（無料）をインストール
2. iOS **16以降**であることを確認（ロック画面ウィジェット対応）

---

## インストール手順

1. **Scriptable** アプリを開く
2. 右上の **「+」** をタップして新規スクリプト作成
3. `jal_award_widget.js` の内容を**全てコピー**してペースト
4. 左上のスクリプト名を **「JAL・ANA予約開始日」** に変更して保存
5. **▶（再生ボタン）** をタップしてプレビューが表示されることを確認

---

## ロック画面への配置（推奨）

1. ロック画面を**長押し** → 「カスタマイズ」 → 「ロック画面」
2. 時計下のウィジェット枠をタップ
3. 「**Scriptable**」を選択
4. **accessoryRectangular**（四角形）サイズを選んで配置
5. 配置後にウィジェットをタップ → `Script: JAL・ANA予約開始日` を選択
6. 「完了」→「設定」で確定

> **表示例**
> ```
> JAL  05/01（金）  0:00
>      深夜0時ジャストに開始
> ANA  04/26（月）  9:30
>      9:30開始（国際線は9:00）
> ```
> ウィジェットをタップするとJALアプリが開きます。

---

## ホーム画面への配置（任意）

1. ホーム画面を**長押し** → 左上の「**+**」
2. 「Scriptable」を選択 → サイズを選んで「ウィジェットを追加」
3. ウィジェットを長押し → 「ウィジェットを編集」
4. Script に **JAL・ANA予約開始日** を選択

> **表示例（small）**
> ```
> 特典航空券 予約開始日
>
> ✈ JAL  360日前  0:00スタート
> 2027/05/01（金）
>
> ✈ ANA  355日前  9:30スタート（国内）
> 2027/04/26（月）
>
> [✈ JAL予約]  [✈ ANA予約]
> ```

## ボタンの動作について

| ボタン | 動作 |
|---|---|
| **✈ JAL予約** | JALアプリを起動（未インストール時はApp Storeへ） |
| **✈ ANA予約** | ANAアプリの予約画面を起動（Universal Link）※ |

※ ANAアプリ未インストール時はブラウザでANAサイトが開きます。
※ ScriptableがWidgetStack.urlに未対応の場合、どちらのボタンもJALアプリが開きます（フォールバック）。

## 定数の変更方法

ルール変更時はスクリプト冒頭の定数を修正するだけです。

```javascript
const JAL_OFFSET_DAYS = 360;  // JAL予約開始日数
const ANA_OFFSET_DAYS = 355;  // ANA予約開始日数
const JAL_START_TIME  = "0:00";   // JAL開始時刻
const ANA_START_TIME  = "9:30";   // ANA開始時刻（国内）
```

## トラブルシューティング

| 症状 | 対処 |
|---|---|
| ウィジェットが「Run Script」のままになる | ウィジェット長押し→編集→Script で該当スクリプトを選択 |
| 日付が更新されない | iOSが更新タイミングを内部制御（数時間に1回）。手動更新はScriptableアプリを開いて▶をタップ |
| ロック画面に表示されない | iOS 16未満の可能性。設定→一般→ソフトウェアアップデートを確認 |
| ANAボタンがJALアプリを開く | ScriptableがWidgetStack.urlに未対応のバージョン。Scriptableを最新版に更新 |
| ANAボタンがブラウザで開く | ANAアプリ未インストール。App StoreからANAアプリをインストール |

---

## 注意事項

- iOSウィジェットは厳密にリアルタイム更新されません。深夜0時の切り替わりに数分〜数十分のラグが発生する場合があります。**実際の予約はJAL・ANA各社サイト/アプリの時計を基準にしてください。**
- 予約日数・開始時刻は変更される可能性があります。変更時はスクリプト冒頭の定数を修正してください。
